# restart-2-uefi
A lightweight Windows batch utility that restarts the system directly into UEFI firmware settings. Includes automatic administrator elevation, UEFI capability detection, a terminal-aware banner, user confirmation prompts, and a simple countdown before initiating the firmware reboot command.

How to Use
1. Download the script
Save restart-to-uefi.bat anywhere on your PC.

2. Double-click to run
The script automatically:
Checks admin rights
Elevates if necessary
Displays the banner
Verifies UEFI support
Prompts for confirmation

3. Press Y to proceed
Then the 5-second COD countdown begins.

4. Your PC reboots directly into UEFI
